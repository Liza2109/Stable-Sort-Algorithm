Sorting Algorithms – Step by Step

Bubble Sort

  1. Compare adjacent elements
  2. Swap if left element is greater
  3. Largest element moves to the end
  4. Repeat for remaining elements
BestAverageWorstSpace
O(n)O(n²)O(n²)O(1)
compare arr[i] and arr[i+1]
if arr[i] > arr[i+1] then swap
move largest element to end
repeat passes until sorted

Insertion Sort

  1. Assume first element is sorted
  2. Pick next element
  3. Shift larger elements
  4. Insert element at correct position
BestAverageWorstSpace
O(n)O(n²)O(n²)O(1)
take first element as sorted
pick next element
shift elements greater than key
insert key at correct position

Merge Sort

  1. Divide array into halves
  2. Recursively sort halves
  3. Merge sorted halves
BestAverageWorstSpace
O(n log n)O(n log n)O(n log n)O(n)
divide array into halves
sort left half
sort right half
merge two sorted halves

Counting Sort

  1. Find maximum and minimum values
  2. Count frequency
  3. Prefix sum
  4. Build output array
BestAverageWorstSpace
O(n+k)O(n+k)O(n+k)O(k)
count[arr[i]]++
prefix sum
build output array
copy output to original array
